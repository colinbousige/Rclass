Reproducible data treatment with R

3 A quick and dirty glance

Here, let’s just see at a glance what we can do with R very quickly, just to keep focused on the end game in case the first chapters feel a bit long… but hey, you gotta learn how to walk before running!

If you have only one thing to remember, loading data from a 2-columns text file and plotting it is as simple as:

df <- read.table("path/to/data.txt", header=TRUE)
plot(df)

Voilà, end of the class.

Of course, you can do so much more…

You can define a vector like so:

A <- seq(from=-pi, to=pi, by=.1) # use the sequence function
B <- sin(A) # apply a function to a vector
C <- c(1, 3, 5, 2) # or specify each values

And then store these variables into a table (a data.frame) with two columns named x and y:

df1 <- data.frame(x=A, y=B)
head(df1)    # glance at the data
##           x             y
## 1 -3.141593 -1.224647e-16
## 2 -3.041593 -9.983342e-02
## 3 -2.941593 -1.986693e-01
## 4 -2.841593 -2.955202e-01
## 5 -2.741593 -3.894183e-01
## 6 -2.641593 -4.794255e-01
names(df1)   # table's column names
## [1] "x" "y"
str(df1)     # table's summary
## 'data.frame':    63 obs. of  2 variables:
##  $ x: num  -3.14 -3.04 -2.94 -2.84 -2.74 ...
##  $ y: num  -1.22e-16 -9.98e-02 -1.99e-01 -2.96e-01 -3.89e-01 ...
summary(df1) # table's stats
##        x           
##  Min.   :-3.14159  
##  1st Qu.:-1.59159  
##  Median :-0.04159  
##  Mean   :-0.04159  
##  3rd Qu.: 1.50841  
##  Max.   : 3.05841  
##        y            
##  Min.   :-0.999574  
##  1st Qu.:-0.696410  
##  Median : 0.000000  
##  Mean   : 0.000111  
##  3rd Qu.: 0.696653  
##  Max.   : 0.999923

Or load data directly from a text file:

# If no header (first line is already data)
df2 <- read.table("Data/rubis_01.txt", header=FALSE)
head(df2) # glance at the data
##         V1       V2
## 1 3063.714 43.91675
## 2 3063.991 47.91675
## 3 3064.267 44.50000
## 4 3064.544 50.50000
## 5 3064.820 50.50000
## 6 3065.097 44.50000
names(df2) <- c("w", "Intensity") # change column names
# If there is a header (first line is column name)
df3 <- read.table("Data/test.dat", header=TRUE)
head(df3, 2) # glance at the data
##   x y
## 1 1 2
## 2 2 3

Plot these:

# specify x and y
plot(A,B)

# or supply a 2-column data.frame
plot(df2, type="l") # type="l" to plot as lines
# add another plot layer
# as lines
lines(df2$w, df2$Intensity/2, col="red")
# as points
points(df2$w, df2$Intensity/3, col="blue")

# histogram plot
hist(df2$Intensity)

And voilà! With this, you can already do a lot of tweaking with your data.

Of course, these plots are ugly and you can do a lot more interesting stuff with R, so let’s get to it!
